# Voxel Star Fighter 🚀

A voxel-style 3D spaceship created in Blender using Python scripting.

![Voxel Star Fighter](voxel_starfighter.png)

## Design Features

- **Central Fuselage**: Sleek main body with rounded nose cone
- **Angled Wings**: Extended side wings with orange accent highlights
- **Twin Engines**: Blue engine nacelles with glow effects
- **Vertical Stabilizers**: Tail fins for stability and style
- **Dark Cockpit**: Detailed pilot canopy

## Color Palette

| Element | Color | Hex |
|---------|-------|-----|
| Body | Steel Grey | `#4C5866` |
| Engines | Electric Blue | `#0066E6` |
| Accents | Bright Orange | `#E66600` |
| Cockpit | Dark Navy | `#1A2633` |

## Technical Details

- **Software**: Blender 4.2.3 LTS
- **Renderer**: Cycles
- **Lighting**: 3-point studio lighting (Key, Fill, Rim)
- **Samples**: 128 samples
- **Resolution**: 1920x1080px
- **Background**: Transparent (alpha channel)
